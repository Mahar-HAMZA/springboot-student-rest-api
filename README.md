# Student Management REST API

This is a simple Spring Boot REST API project built for learning backend development.

## 🚀 Features

- Add a student using POST request
- View all students using GET request
- In-memory data storage using LinkedList
- Tested using Postman

## 🛠 Technologies Used

- Java
- Spring Boot
- REST API
- Postman

## 📌 API Endpoints

### Add Student

POST /linkWork

Sample JSON Body:

{
  "name": "Hamza",
  "age": 19,
  "grade": "A"
}

---

### Get All Students

GET /linkWork

---

## ▶ How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the main class
4. Test endpoints using Postman at:

http://localhost:8080/linkWork

---

## 📚 Learning Purpose

This project was built to understand:

- REST Controllers
- RequestBody
- JSON to Object mapping
